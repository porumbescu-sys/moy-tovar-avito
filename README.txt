Мой Товар — Streamlit v3 (компактный рабочий интерфейс)

Как запустить локально:
1. Установите Python 3.10+
2. Откройте терминал в папке проекта
3. Выполните:
   pip install -r requirements.txt
   streamlit run streamlit_app.py

Для Streamlit Community Cloud:
1. Загрузите содержимое этой папки в GitHub-репозиторий
2. На streamlit.app создайте приложение
3. Main file path: streamlit_app.py

Что умеет:
- загрузка Excel/CSV
- поиск по артикулу, названию, бренду
- поиск по нескольким артикулам сразу
- выбор главной цены: Продажа / -12% / -20%
- округление выбранной цены вверх до 100
- кнопка копирует именно выбранную цену
- экспорт найденного в Excel
