Мой Товар — красивая Streamlit-версия

Как запустить локально:
1. Установите Python
2. В папке проекта выполните:
   pip install -r requirements.txt
   streamlit run streamlit_app.py

Как разместить на streamlit.app:
1. Загрузите содержимое этой папки в GitHub-репозиторий
2. На Streamlit Community Cloud выберите репозиторий
3. Main file path: streamlit_app.py
4. Deploy

Что умеет:
- загрузка Excel/CSV
- поиск по артикулу / названию / бренду
- кнопка Найти
- кнопка Очистить
- выбор цены: Продажа / -12% / -20%
- округление выбранной цены вверх до 100
- копирование именно выбранной цены
- экспорт результата в Excel
