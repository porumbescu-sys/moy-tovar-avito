Мой Товар — Streamlit ready

Что внутри:
- streamlit_app.py
- requirements.txt
- .streamlit/config.toml
- data/demo_price.xlsx
- run_local.bat

Как запустить локально:
1) Установите Python 3.10+
2) Откройте терминал в папке проекта
3) Выполните:
   pip install -r requirements.txt
   streamlit run streamlit_app.py

Как разместить на streamlit.app:
1) Загрузите содержимое папки в GitHub-репозиторий
2) Откройте Streamlit Community Cloud
3) New app / Create app
4) Укажите репозиторий и main file path: streamlit_app.py
5) Deploy
